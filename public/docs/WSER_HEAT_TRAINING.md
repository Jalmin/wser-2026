# WSER 2026 — Heat Training

*Protocole d'acclimatation chaleur pour les canyons (35-45°C)*

---

## Le Problème WSER

### Températures par section

| Section | Km | Température typique | Heure de passage (sub-20h) |
|---------|-----|---------------------|---------------------------|
| High Country | 0-50 | 5-15°C (matin) | 5am-11am |
| **Canyons** | 50-100 | **35-45°C** | **11am-6pm** |
| California Loop | 100-128 | 25-35°C | 6pm-11pm |
| Final Push | 128-161 | 15-25°C (nuit) | 11pm-1am |

**Le piège** : Tu arrives aux canyons (km 50) en milieu de journée, après déjà 6-7h d'effort. C'est là que la chaleur frappe — au moment où tu es le moins frais.

Sans acclimatation, les conséquences sont prévisibles :
- FC qui explose (+20-30 bpm à effort égal)
- Sudation inefficace
- Risque hyperthermie, crampes, DNF

### Ton gap (analyse Strava)

**Aucune trace de heat training dans ton historique.** C'est le gap le plus critique à combler.

## Physiologie de l'Acclimatation

### Les 5 adaptations clés

| Adaptation | Délai | Mécanisme | Bénéfice course |
|------------|-------|-----------|-----------------|
| **Volume plasmatique** | 3-5 jours | Rétention fluides, expansion plasma | Meilleure thermorégulation |
| **Sudation précoce** | 5-7 jours | Abaissement seuil de déclenchement | Refroidissement plus rapide |
| **Débit sudoral** | 7-10 jours | +10-20% de sueur/heure | Évacuation chaleur accrue |
| **FC réduite** | 7-10 jours | -10-15 bpm à effort égal | Moins de stress cardiovasculaire |
| **Concentration sueur** | 10-14 jours | -30-40% NaCl dans sueur | Moins de perte de sel |

### Timeline d'adaptation

```
Jour 1-3   : Volume plasmatique ↑ (+10-15%)
Jour 3-5   : Début sudation plus précoce
Jour 5-7   : Débit sudoral ↑, FC commence à baisser
Jour 7-10  : FC stabilisée (-10-15 bpm), sudation efficace
Jour 10-14 : Sueur plus diluée, adaptation "complète"
```

### Demi-vie de perte

**Sans exposition** : Les adaptations se perdent à ~2.5%/jour.

| Temps sans heat | Perte estimée |
|-----------------|---------------|
| 1 semaine | -17% |
| 2 semaines | -35% |
| 3 semaines | -50% |
| 4 semaines | -65% |

**Bonne nouvelle** : La "mémoire physiologique" permet une réadaptation plus rapide (5-7 jours vs 10-14 jours initial). C'est pourquoi le re-boost J-10→J-3 fonctionne.

## Phase 1 : Maintenance (janvier → mi-avril)

### Objectif

Maintenir une base d'acclimatation minimale pendant l'hiver. Pas d'adaptation majeure attendue, juste ne pas partir de zéro en mai.

### Protocole

| Paramètre | Valeur |
|-----------|--------|
| Fréquence | 1×/semaine |
| Durée | 45-60 min |
| Méthode | Vélo indoor (home trainer) |
| Tenue | K-way fermé + bonnet |
| Intensité | Z2 (conversation possible) |
| Température pièce | 22-25°C si possible |

### Setup indoor

1. **Home trainer** dans pièce chauffée
2. **K-way fermé** — pas de fermeture ouverte, effet sauna
3. **Bonnet** — 40% de la chaleur s'échappe par la tête
4. **Ventilateur OFF**
5. **Pas de douche froide après** — laisser le corps redescendre naturellement (30min)

### Pourquoi vélo et pas course

- Pas d'impact = moins de stress musculaire
- Position statique = chaleur plus concentrée
- Effort contrôlable précisément
- Pas de risque de coup de chaleur dehors

## Phase 2 : Bloc Maroc (fin avril)

### Objectif

**Initiation massive** — Lancer les adaptations plasma/sudation avec 7-10 jours d'exposition consécutive au soleil naturel.

### Pourquoi c'est supérieur

L'exposition soleil naturel combine :
- Chaleur ambiante
- Rayonnement solaire direct (infrarouge)
- Humidité variable
- Composante psychologique (confiance)

Les études montrent une adaptation 20-30% plus rapide en exposition naturelle vs indoor.

### Protocole Maroc

| Paramètre | Valeur |
|-----------|--------|
| Durée séjour | 7-10 jours minimum |
| Sessions/jour | 1-2 |
| Horaire optimal | 11h-14h (pic chaleur) |
| Durée session | 1h-1h30 |
| Activité | Run facile, parapente, marche |
| Intensité | Basse (Z2 max) |

### Format session type

1. **Départ** : 11h-12h
2. **Activité** : Run facile ou marche rapide
3. **Durée** : 1h-1h30 selon acclimatation
4. **Intensité** : Conversation facile (Z2)
5. **Post-session** : Pas de refroidissement brutal. Attendre 30min avant douche/piscine.

### Core temp cible

| Jour | Core temp cible | Durée maintien |
|------|-----------------|----------------|
| J1-J3 | 38.0-38.3°C | 30-45min |
| J4-J7 | 38.3-38.5°C | 45-60min |
| J8-J10 | 38.5-38.8°C | 60min+ |

### Signaux d'adaptation

- Sudation plus précoce (tu commences à suer plus tôt)
- Sueur moins salée (goût moins prononcé)
- FC plus basse à effort égal
- Sensation de "gérer" la chaleur

## Phase 3 : Intensive (mai → mi-juin)

### Objectif

**Maximiser l'acclimatation** avant WSER. Objectif : core temp 38.5-39°C maintenue 45min+ par session.

### Protocole

| Paramètre | Valeur |
|-----------|--------|
| Fréquence | 4-5×/semaine |
| Durée | 60-90 min |
| Méthode | Mix indoor + outdoor |
| Core temp cible | 38.5-39°C pendant 45min+ |

### Semaine type

| Jour | Session | Méthode |
|------|---------|---------|
| Lundi | Repos | - |
| Mardi | **Heat run** | Course 12h-14h, 60-75min |
| Mercredi | Séance signature | (pas heat) |
| Jeudi | **Heat indoor** | Vélo K-way, 60min |
| Vendredi | Muscu | (pas heat) |
| Samedi | **Longue heat** | Départ 10h, finir heures chaudes |
| Dimanche | **Heat run** ou récup | Course 12h-14h ou récup |

### Options selon météo

| Température extérieure | Méthode |
|------------------------|---------|
| >30°C | Run normal, pas de couche supplémentaire |
| 25-30°C | Run avec une couche sombre |
| 20-25°C | Run avec deux couches ou K-way léger |
| <20°C | Indoor obligatoire |

### Longue en chaleur

- Départ 10h-11h pour finir aux heures chaudes
- Les dernières 2-3h en pleine chaleur
- Simule l'arrivée aux canyons WSER fatiguée + chaleur

### Objectif

**Réactiver la mémoire physiologique** avant la course. Les adaptations perdues pendant le taper se récupèrent plus vite.

## Marqueurs de Progression

### Comment savoir que ça fonctionne

| Marqueur | Avant acclimatation | Après acclimatation |
|----------|---------------------|---------------------|
| FC à effort Z2 | 145-155 bpm | 130-140 bpm |
| Début sudation | Après 10-15min | Après 3-5min |
| Sensation subjective | "J'étouffe" | "C'est chaud mais gérable" |
| Récup post-heat | 2-3h | 30-60min |
| Couleur urine | Foncée | Claire |

### Test de validation

**Avant THP 120 (mi-mai)** : Faire une séance de 60min à 12h-14h. Si FC reste <145 bpm et sensation "gérable", l'acclimatation est en bonne voie.

## Conseils Pratiques

### Setup indoor optimal

- **Pièce** : 22-25°C minimum, fenêtres fermées
- **Ventilo** : OFF (sinon évaporation trop efficace)
- **Tenue** : K-way imperméable fermé + bonnet ou capuche
- **Hydratation** : 500-750ml/h pendant la séance
- **Post-session** : Pas de douche froide pendant 30min

### Erreurs courantes

1. **Douche froide immédiate** : Stoppe brutalement le stimulus. Attendre 30min.

2. **Intensité trop haute** : Le but est la chaleur, pas l'effort. Z2 max.

3. **Sessions trop courtes** : <40min = stimulus insuffisant.

4. **Irrégularité** : 1 session/semaine en phase intensive = insuffisant.

5. **Arrêt trop tôt** : Stop à J-10 au lieu de J-3 = perte d'adaptation.

### Hydratation pendant sessions

| Durée session | Hydratation |
|---------------|-------------|
| 45min | 500ml |
| 60min | 600-750ml |
| 90min | 1000-1200ml |

Ajouter électrolytes si >60min.

## Intégration au Planning

### Janvier-Février (Phase BASE)

- 1×/semaine indoor
- Placer le dimanche ou jour de récup
- Ne pas cumuler avec séance dure

### Mars-Avril (Phase DÉVELOPPEMENT)

- 1-2×/semaine
- Bloc Maroc fin avril = priorité absolue
- Si pas Maroc : 2×/sem indoor + runs midi week-end

### Mai-Juin (Phase SPÉCIFIQUE)

- 4-5×/semaine
- Runs midi quand météo le permet
- Longues aux heures chaudes
- THP 120 = test grandeur nature

### Taper (J-14 → J-0)

- Re-boost J-10 → J-3
- Stop J-3
- Ne pas stresser si conditions Californie différentes
